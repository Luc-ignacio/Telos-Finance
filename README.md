## ğŸš€ Features

- **Wallet Management**
  - Create and manage multiple wallets
  - Default wallet automatically created for new users
  - Portfolio-level summary and performance overview

- **Asset System**
  - Centralised asset catalogue (e.g. stocks, crypto, ETFs, bonds)
  - Assets reusable across multiple wallets
  - Supports multiple asset classes

- **Holdings & Transactions**
  - Track holdings per wallet and asset
  - Buy and sell transactions
  - Automatic average price calculation
  - Quantity and cost basis tracking

- **Performance Tracking**
  - Unrealised and realised gains
  - Portfolio allocation insights
  - Foundation for future analytics and charts

- **Scalable Architecture**
  - Clear separation between assets, wallets, holdings, and transactions
  - Designed to grow into a full-featured finance platform
